# SwiftLink URL Shortener

SwiftLink is a modern and responsive URL shortener website built with HTML5, CSS3, and Vanilla JavaScript.  
It uses a real public API to shorten links and features a clean glass style user interface.

## Features

* Fast URL shortening using a public API
* Glass style modern user interface
* Fully responsive design for mobile, tablet, and desktop
* Copy to clipboard support
* URL validation before submit
* Loading spinner during processing
* Clean and simple code structure

## Tech Stack

* HTML5
* CSS3
* Vanilla JavaScript
* CleanURI Public API

## Project Structure

* index.html  
  Contains HTML, CSS, and JavaScript in one file

You can split the code later into separate files if needed.

## How It Works

1. User pastes a long URL into the input field
2. JavaScript validates the URL format
3. Request is sent to the CleanURI API
4. API returns a shortened link
5. Result is shown with a copy button

## API Used

CleanURI URL Shortener  
Endpoint  
https://cleanuri.com/api/v1/shorten

Method  
POST

## How To Use

1. Download or copy the index.html file
2. Open it in any modern browser
3. Paste a long URL
4. Click the Shorten button
5. Copy the shortened link
